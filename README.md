# Does-Twitter-Hate-Cats
This project uses AWS services to gauge social media sentiment. The tongue-in-cheek example here is whether or not Twitter hates cats.

## Technology used

Twitter Developer API
API Gateway
AWS Comprehend
DynamoDB
Lambda
S3
